# c_physics_engine

A custom physics engine written in C/C++

## Reguirements

- cmake
- make
- gcc/clang
- clang-format
- doxygen

## Makefile

- `make clean`: delete files ignored by git
- `make fmt`: format all code
- `make test`: test the lib

- `make doc`: generate documantation
- `make build`: build the lib\
